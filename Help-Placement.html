ETools.Placement - Help

Place Between Two Points

This tool allows you to place a selected family instance at the midpoint between two picked points in the model.

  • Start the command from the ETools → Placement panel.
  • Select the desired family type in the Revit ribbon or Properties palette.
  • Click on two points in the model. The object will be placed at the exact midpoint.
  • Works with face-based and non-hosted families.
Note: The family must be placeable manually and should not require a host if not face-based.

Place Array Between Two Points

This tool allows you to create a rectangular array of a selected family between two picked points, defining the number of rows and columns.

  • Start the command from the ETools → Placement panel.
  • Select the desired family type before launching the command.
  • Pick two points in the model — these define the rectangular area for placement.
  • Enter the number of rows and columns in the input window.
  • The tool calculates spacing and places instances evenly within the rectangle.
Note: The spacing is automatically computed based on the distance between picked points and the number of instances.

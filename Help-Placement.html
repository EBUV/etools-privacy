ETools.Placement - Help

Place Between Two Points

This tool allows you to place a selected family instance at the midpoint between two picked points in the model.

  • You can either pre-select the object to be copied or launch the command and then select it when prompted.
  • After selecting the source element, pick two points in the model. The new instance will be placed at the midpoint.
  • Works with face-based and non-hosted families.
Note: The selected object must be a family instance that can be placed manually. Face-based families require a valid face during placement.

Place Array Between Two Points

This tool allows you to create a rectangular array of a selected family instance between two picked points, with a specified number of rows and columns.

  • You can pre-select the source object or select it after launching the command.
  • After starting the command, you will be asked to enter the number of rows and columns.
  • Then, pick two points in the model. These define the rectangular area for the array.
  • The tool calculates the spacing and places the copies evenly across the area, using a margin of x ⁄ 2 from the edges and a distance of x between the elements.
Note: The original object will not be moved or deleted. Spacing is automatically calculated based on the total area and instance count.

Privacy Policy for ETools Revit Add-in

This Revit add-in (“ETools”) is developed and published by Evgenii Burlakov.

1. Data Collection

This app does not collect, store, transmit, or process any personal data, user data, or project data. All operations are executed locally within Autodesk Revit.

2. Third Parties

ETools does not interact with any third-party services, analytics tools, or SDKs. No data is shared externally.

3. Data Retention

Since no data is collected, there is no data retained. The app operates only during active Revit sessions and performs localized model manipulations.

4. User Rights and Consent

No data is stored or retained by the app. Users do not need to provide consent for data usage. If at any time the policy changes, updated versions will be made available via the official distribution page.
